What is Stream Companion?

Stream Companion is a free open source, fully responsive managing application for your series, movies and other things that you watch.

The only requirement for this is .NET 4.0 from Microsoft, get it here.

Easy to use!

Self explaning usage. But if you want to read more or get stuck sometimes check out the usage guide.
